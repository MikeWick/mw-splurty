# Splurty Quote Generator

> A Ruby on Rails app that allows a user to contribute a quote and author to be input and saved in the database.

## Features
* New Quote submission form
* Database to store quotes and authors

## Technologies
* Ruby on Rails
* JavaScript
* HTML
* CSS
